# nonebot_plugin_osumania_toolkit
this is a nonebot plugin with some useful osu!mania utilities

This plugin is under development, so there will be no releases.
If you want to install it, try to clone this repo and [load it as a plugin](https://nonebot.dev/docs/2.4.2/tutorial/create-plugin#%E5%8A%A0%E8%BD%BD%E6%8F%92%E4%BB%B6).
